# User Story 01: Thêm nhà vận chuyển mới

## Thông tin

- **Feature**: Example Feature - Add Carrier
- **Story ID**: US-01
- **Priority**: High
- **Status**: Draft

## User Story

**As a** quản trị viên hệ thống,
**I want** thêm nhà vận chuyển mới vào hệ thống,
**So that** tôi có thể mở rộng lựa chọn vận chuyển cho khách hàng.

## Acceptance Criteria

1. **AC-01**: Admin có thể nhập thông tin nhà vận chuyển (tên, mã, API endpoint)
2. **AC-02**: Hệ thống validate thông tin trước khi lưu
3. **AC-03**: Sau khi thêm thành công, nhà vận chuyển xuất hiện trong danh sách
4. **AC-04**: Hệ thống gửi notification xác nhận

## Notes

- Cần tích hợp API của nhà vận chuyển bên thứ 3
- Xem thêm specs và plans trong folder tương ứng
