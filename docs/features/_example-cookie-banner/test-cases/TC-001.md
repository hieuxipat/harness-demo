## Preconditions
- Database có bảng settings
- Cookie banner đang disabled

## Steps
1. POST /api/settings/cookie-banner với body `{ "enabled": true }`
2. GET /api/settings/cookie-banner

## Expected Result
- POST trả về 200 với `{ "enabled": true }`
- GET trả về `{ "enabled": true }` (persist đúng)

## Actual Result
Pass — Settings saved and persisted correctly. Response time < 50ms.
