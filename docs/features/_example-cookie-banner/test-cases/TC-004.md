## Preconditions
- Cookie banner enabled
- Default title và body text

## Steps
1. PATCH /api/settings/cookie-banner với `{ "title": "Custom Title", "body": "Custom body text" }`
2. Verify response 200
3. Mở storefront → verify banner hiển thị "Custom Title" và "Custom body text"

## Expected Result
- API trả về 200 với nội dung đã cập nhật
- Storefront banner hiển thị đúng nội dung tuỳ chỉnh

## Actual Result
Pass — Custom content displayed correctly on storefront within 2s of save.
