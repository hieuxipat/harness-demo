## Context
Cần lưu lựa chọn cookie consent của khách hàng. Có 2 phương án: lưu server-side (database) hoặc client-side (localStorage).

## Decision
Lưu consent trong **localStorage** của browser.

## Consequences
- **Pros:** Không cần API call, hoạt động offline, GDPR-compliant (data ở client), giảm load server
- **Cons:** Mất consent khi clear browser data, không sync across devices

## Alternatives Considered
1. **Server-side (database)** — Cần user login, phức tạp hơn, tốn API call mỗi page load
2. **Cookie-based** — Ironic (dùng cookie để track cookie consent), size limit 4KB
