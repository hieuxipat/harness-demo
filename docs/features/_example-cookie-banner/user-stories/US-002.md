## User Story
Là **khách hàng**, tôi muốn **chọn chấp nhận hoặc từ chối cookies** để **kiểm soát quyền riêng tư của mình**.

## Acceptance Criteria
- [x] AC-1: Banner hiển thị 2 nút: "Accept All" và "Reject All"
- [x] AC-2: Lựa chọn được lưu vào localStorage, không hỏi lại khi quay lại
- [x] AC-3: Nếu reject, chỉ load essential cookies (không load tracking/analytics)

## Technical Notes
- Storefront: Banner component với Accept/Reject buttons
- localStorage key: `cookie_consent` = `accepted` | `rejected`
- Script loader check consent trước khi inject tracking scripts
